Featured Projects

GemXcel — AI-Integrated Education Platform

Solo Developer · Research & Development

GemXcel is an advanced education platform I designed and built from scratch to demonstrate the effectiveness of combining AI with Gamification in learning. The system integrates Supabase for database management, solving data synchronization challenges between local and server storage, and ensuring platform stability at real-world scale.

I built the system architecture using API calls to automate text content extraction and generate academic assessments — minimizing question preparation time and personalizing each user's learning experience. The project is fully managed through GitHub and deployed on Cloudflare Pages.
